Privacy Policy

Last updated: March 29, 2026

No data collection

Cortex does not collect, transmit, or store any personal data. There is no telemetry, no analytics, and no network calls made by the plugin.

How Cortex works

All coordination happens through local markdown files in your team directory — a folder on your machine that you control. Agent notes, project notes, and work queues are plain text files that never leave your filesystem unless you choose to share them (e.g., via git).

Configuration

Cortex stores configuration at ~/.cortex/config.yaml on your local machine. This file may contain a Telegram chat ID if you choose to configure Telegram integration. This data stays on your machine and is never sent to agentweave or any third party.

Third-party services

If you configure Telegram integration, messages are sent through Telegram's API via the Claude Code Telegram plugin — not through Cortex. Cortex itself makes no network requests.

Open source

Cortex is fully open source under the MIT License. You can inspect every line of code at github.com/agentweave/cortex.

Contact

Questions about this policy? Open an issue at github.com/agentweave/cortex/issues.
